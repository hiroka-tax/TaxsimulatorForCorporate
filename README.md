# 法人税・消費税シミュレーター

売上と利益率を入力するだけで、法人税等・消費税の概算納税額を手軽に確認できるWebツールです。

## 概要

税理士向け・経営者向けに、法人の税負担をざっくり把握するためのシミュレーターです。  
ブラウザ上で完結するため、インストール不要でそのままご利用いただけます。

## 主な機能

- **売上・経費の入力** — 売上金額と経費金額を入力し、利益率／経費率を自動計算して表示
- **法人税等の計算** — 法人税・地方法人税・事業税・特別法人事業税・住民税（法人税割・均等割）を自動計算
- **消費税の計算** — 簡易課税（みなし仕入率選択）または2割特例に対応
- **中間納付の控除** — 法人税等・消費税それぞれの中間納付額を入力し、期末納付額を算出
- **繰越欠損金の控除** — 繰越欠損金を入力することで課税所得を調整

## 使い方

1. `index.html` をブラウザで開く
2. 売上（円）を入力する
3. 経費（円）を入力する（利益率・経費率は自動表示）
4. 繰越欠損金・中間納付額など該当する項目を入力する
5. 簡易課税の有無・みなし仕入率・2割特例を選択する
6. 「計算結果」欄に各税目の概算額が自動表示される

## 税率・計算の前提

| 項目 | 税率・内容 |
|------|-----------|
| 法人税（課税所得800万円以下） | 15% |
| 法人税（課税所得800万円超） | 23.2% |
| 地方法人税 | 法人税額 × 10.3% |
| 事業税（所得割） | 3.5% |
| 特別法人事業税 | 事業税額 × 37% |
| 住民税（法人税割） | 法人税額 × 7% |
| 住民税（均等割） | 70,000円（固定） |
| 消費税（簡易課税） | 売上消費税額 ×（1 − みなし仕入率） |
| 消費税（2割特例） | 売上消費税額 × 20% |

> **注意**: 本ツールは概算計算を目的としています。正確な税額については税理士にご相談ください。
